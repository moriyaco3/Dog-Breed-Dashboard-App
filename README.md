# Dog Breed Dashboard

An interactive data exploration app for dog breeds, built with **Python** and **Streamlit**, using data from **TheDogAPI**.

The app is designed for people who are interested in dogs and want to compare breeds based on size, life span, temperament, and other characteristics â€” especially useful for people looking to choose a dog.

---

## Project Description

This project visualizes and explores dog breed data using an external API.  
It includes:

- A **gallery view** of dog breeds with images
- **Detailed breed pages** with physical and historical information
- A **Statistics section** with interactive charts
- A **Data page** that shows the raw dataset in table form

The focus is on making the data easy to understand and useful for real-world decision making.
